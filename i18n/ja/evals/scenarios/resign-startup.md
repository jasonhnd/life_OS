# シナリオ：退職して起業

**パス**: ⚖️ フルコート

## ユーザーメッセージ

```
東京の日本企業でソフトウェアエンジニアをしています。年収800万円で、勤続3年になります。最近どんどん退屈に感じるようになり、辞めて自分のSaaSプロダクトを作りたいと思っています。貯金は約500万円あります。共同創業者はいませんし、何のプロダクトを作るかもまだ決めていません — ただ、若いうちにチャレンジすべきだと感じています。
```

## 期待される動作

- **Router**: 上奏すべき（キャリア + 財務 + 不可逆な変化に関わる）、直接処理すべきではない
- **Planner**: Six Domains すべてを起動すべき、少なくとも財務/能力/人脈/実行/リスク/ライフスタイルの6つの軸をカバー
- **Reviewer の計画審査**: 「何のプロダクトを作るか決めていない」を重大な欠陥として指摘すべき
- **FINANCE ドメイン**: 500万円 / 東京の月間生活費 ≈ 20〜25ヶ月の資金余力、ただし起業コストの控除が必要
- **GOVERNANCE ドメイン**: 競業避止条項（日本企業では一般的）、ビザへの影響（就労ビザの場合）を確認すべき
- **EXECUTION ドメイン**: 「具体的な方向性がない」ことを最大の実行障害として指摘すべき
- **INFRA ドメイン**: 起業期間中の心身の健康と生活の質について言及すべき
- **Reviewer の最終審査**: スコアがすべて >= 7 にはならないはず（「何を作るか決めていない」は明確な弱点）
- **Advisor**: 「若いうちにチャレンジすべき」が認知バイアスでないか検証すべき

## 品質チェックポイント

- [ ] Router が直接処理せず、正しく上奏した
- [ ] Planner が5つ以上のドメインを起動した
- [ ] Reviewer が少なくとも1つの項目に疑問を呈した
- [ ] FINANCE ドメインが具体的な資金余力の数字を提示した
- [ ] GOVERNANCE ドメインが競業避止条項またはビザの問題に言及した
- [ ] 少なくとも1つのドメインのスコアが < 6
- [ ] アクション項目に「まずプロダクトの方向性を決定する」が含まれている
- [ ] Advisor が実質的な進言を行った（一般論ではなく）
- [ ] Auditor が少なくとも1つの改善点を特定した
