# シナリオ：高額購入

**パス**: 🏃 快車道分析

## ユーザーメッセージ

```
MacBook Pro M4 Maxを買いたいと思っています。フルスペックだと約60万円です。今は3年前のM1 MacBook Airを使っていて、普段の開発には問題ないのですが、AIモデルの実行には少し遅いです。主な理由は、同僚がみんな新モデルに買い替えたのを見て、自分もアップグレードすべきだと感じたことです。
```

## 期待される動作

- **丞相**: 上奏すべき（金額 > 月収の10%）、戸部 + 兵部 + 刑部の起動を推奨
- **中書省**: 六部すべてを起動すべきではない（対人関係/学習/健康に関わらない）、3つの六部で十分
- **戸部**: 60万円の収入/貯蓄に対する比率を定量化し、財務的妥当性を評価すべき
- **兵部**: M1 Air vs M4 Maxの実際の性能差を評価し、よりコストパフォーマンスの良い選択肢がないか検討すべき
- **刑部**: 「同僚がみんな買い替えた」がFOMO（取り残される恐怖）でないか確認し、衝動購入のリスクを評価すべき
- **門下省の感情面レビュー**: 「同僚がみんな買い替えたのを見て」という動機に疑問を呈すべき
- **諫官**: 同調行動のシグナルを指摘すべき

## 品質チェックポイント

- [ ] 丞相が直接「買え」または「買うな」と言わず、正しく上奏した
- [ ] 中書省が六部すべてを起動していない（起動数 <= 4）
- [ ] 戸部が具体的な財務比率を提示した
- [ ] 兵部が実際のニーズ vs 欲求のギャップを分析した
- [ ] 刑部または門下省がFOMO / 同調行動の動機を特定した
- [ ] 全体スコアが > 8 にならないはず（明確な非合理的動機が存在）
