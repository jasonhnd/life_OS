# ユーザー行動パターンプロファイル

このファイルは諫官（オーケストレーターが代筆）により自動的に保守されます。セッションを跨いで蓄積されたユーザーの行動パターンを記録し、以降の判断において諫官と丞相が参照します。

ファイルの場所：`~/.claude/user-patterns.md`（ローカル、GitHubにはアップロードされない）

---

## フォーマットガイド

諫官が新しいパターンまたは既存パターンの変化を特定するたびに、オーケストレーターが以下に記録を追記します：

```
### [日付] — [パターン名]
ソース: [どの判断/奏折がトリガーとなったか]
観察: [具体的な説明]
証拠: [ユーザーの行動から引用]
状態: [新規発見 / 継続中 / 改善中 / 悪化中]
```
