# 六部職能と四司の詳細

中書省が立案時にこのファイルを参照する。

## 👥 吏部・人の管理
- **選賢司**：人を見極める、パートナー/投資家/クライアントの評価、メンター関係の構築
- **考功司**：関係の健全性評価、社交ROI、有害な関係の特定
- **封賞司**：関係の育成、互恵的な好意、贈答戦略、重要な日付のリマインダー
- **調配司**：チームビルディング、タスク委任、家族の分担、アウトソーシング/採用判断

## 💰 戸部・金銭の管理
- **収入司**：収入構造、給与交渉、副業、パッシブインカムチャネル
- **度支司**：予算管理、月次支出、支出習慣、サブスクリプション整理
- **資産司**：投資ポートフォリオ、暗号資産、不動産、市場リスク
- **儲備司**：緊急資金（6-12ヶ月）、保険、税務計画、退職計画

## 📖 礼部・秩序とイメージの管理
- **科挙司**：学習ロードマップ、スキル開発、資格試験、ナレッジマネジメント、読書計画
- **典儀司**：パーソナルブランド、社交イメージ、外見とスタイル、LinkedIn
- **文翰司**：ブログ/ニュースレターのコンテンツ計画、ライティングスタイル、スピーチ準備
- **外交司**：異文化コミュニケーション（ユーザーの言語）、ビジネスエチケット、ネットワーキング、面接での表現力

## ⚔️ 兵部・行動の管理
- **軍令司**：プロジェクト計画、タスク分解、優先順位付け、期限、進捗追跡
- **装備司**：ツール選定、ハードウェアセットアップ、ソフトウェアワークフロー、開発環境
- **情報司**：業界調査、競合分析、市場インテリジェンス、技術トレンド
- **後勤司**：エネルギー管理、ワークフロー最適化、会議効率、先延ばし管理

## ⚖️ 刑部・ルールの管理
- **律法司**：法的リスク、契約レビュー、知的財産、税務コンプライアンス、競業避止制限
- **審計司**：判断レビュー、時間監査、投入対産出の振り返り、失敗分析
- **糾察司**：悪習慣の特定/是正、コミットメント追跡、目標実行率、自己欺瞞の検出
- **防御司**：情報セキュリティ、プライバシー保護、詐欺の特定、心理的防御（操作/ガスライティング）

## 🏗️ 工部・建設の管理（健康を含む）
- **体健司**：運動計画、食事と栄養、睡眠の質、健康診断、メンタルヘルス
- **営建司**：住空間の最適化、引っ越し、リノベーション、ワークスペースレイアウト
- **数造司**：Notion/ナレッジベース、サーバーとドメイン、開発環境、データバックアップ、自動化
- **水利司**：日常ルーティン、朝/就寝前の手順、家事効率、生活リズム

## 管轄境界ルール

以下のルールは重複する領域での担当を決定する。原則：**表面的症状ではなく、根本原因で分類する。**

### 「最近すごく疲れている」 — 兵部/後勤 vs 工部/体健 vs 工部/水利

| シナリオ | 担当 | 根拠 |
|----------|-------|-----------|
| 「3週間連続で残業、完全に消耗した」 | 工部/体健（主管）+ 兵部/後勤（補佐）| 根本原因は肉体的疲弊。兵部はスケジュール最適化で補佐 |
| 「毎日夜遅くまで忙しいが振り返ると何も実質的にやっていない」 | 兵部/後勤 | 根本原因はエネルギー配分とワークフロー非効率 |
| 「朝起きれない、夜寝れない、リズムが完全に崩壊」 | 工部/水利（主管）+ 工部/体健（補佐）| 根本原因はルーティンの崩壊 |
| 「不安で眠れない、日中はエネルギーがない」 | 工部/体健 | 根本原因はメンタルヘルス |

**クイック判断ルール：身体の不調 → 体健、リズム崩壊 → 水利、仕事の非効率 → 後勤。**

### 「ずっと先延ばししている」 — 兵部/後勤 vs 刑部/糾察 vs 工部/体健

| シナリオ | 担当 | 根拠 |
|----------|-------|-----------|
| 「タスクが大きすぎてどこから手をつけていいか分からない」 | 兵部/後勤 | 根本原因はタスク未分解、開始戦略の欠如 |
| 「やるべきだと分かっているのにスマホを手に取って2時間スクロール」 | 刑部/糾察（主管）+ 兵部/後勤（補佐）| 根本原因は悪習慣。後勤は環境設計で補佐 |
| 「会議前にいつも詰め込み、もう5回目」 | 刑部/糾察 | 根本原因はコミットメントの繰り返し違反 |
| 「午後のエネルギー低下時にいつも重要タスクを先延ばし」 | 兵部/後勤 | 根本原因はエネルギー管理とスケジューリングの不整合 |
| 「そのタスクを考えるだけで不安になり先延ばし続ける」 | 工部/体健 | 根本原因は心理的障壁（恐怖/不安）|

**クイック判断ルール：やり方が分からない → 後勤（方法の問題）、やりたくない → 糾察（習慣の問題）、やる勇気がない → 体健（心理的問題）。**

### 「贈り物はどうすべき？」 — 吏部/封賞 vs 礼部/外交

| シナリオ | 担当 | 根拠 |
|----------|-------|-----------|
| 「彼女の誕生日、ポイントの高いプレゼントは？」 | 吏部/封賞 | 核心は関係の育成 |
| 「日本のクライアントが来社、手土産のプロトコルは？」 | 礼部/外交（主管）+ 吏部/封賞（補佐）| 核心は異文化ビジネスエチケット |
| 「お世話になった元上司にお礼がしたい、どう感謝を示すか」 | 吏部/封賞 | 核心は互恵的好意 |
| 「パートナーの年次イベントに初参加、手土産は必要？」 | 礼部/外交 | 核心は場のエチケット規範 |

**クイック判断ルール：人への心からの贈り物 → 吏部（関係戦略）、場にふさわしい贈り物 → 礼部（エチケット規範）。**

### 補佐メカニズム

シナリオが「補佐」とマークされている場合：
- **主管部**が提案の作成と判断責任を負う
- **補佐部**が専門的インプットを提供し、主管報告の専用セクションに埋め込む
- 主管と補佐が意見不一致の場合、政事堂に上程して解決
