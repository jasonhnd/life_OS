# 標準シナリオ推奨構成

## 1. キャリア転換（退職/転職/起業）
財務 → 戸部 | スキル → 礼部 | ネットワーク → 吏部 | 実行 → 兵部 | リスク → 刑部 | 生活への影響 → 工部
門下省の焦点：逃避なのか追求なのか？家族はどう感じている？

## 2. 投資判断
財務 → 戸部 | 市場 → 兵部 | リスク → 刑部 | 人脈 → 吏部
門下省の焦点：FOMOか合理的か？全損しても生きていけるか？

## 3. 転居/引っ越し
財務 → 戸部 | 人間関係 → 吏部 | インフラ → 工部 | コンプライアンス → 刑部 | 文化 → 礼部
門下省の焦点：行き先を本当に知っているか？引き返すコストは？

## 4. 年間目標計画
人間関係 → 吏部 | 財務 → 戸部 | 学習 → 礼部 | 実行 → 兵部 | 是正 → 刑部 | 健康 → 工部
門下省の焦点：目標が多すぎないか？測定可能か？価値観と整合しているか？

## 5. プロダクト/スタートアップ判断
市場 → 兵部 | 財務 → 戸部 | チーム → 吏部 | コンプライアンス → 刑部 | インフラ → 工部 | ブランド → 礼部
門下省の焦点：本当のペインポイントを解決しているか？自分が適任か？

## 6. 大きな買い物
財務 → 戸部 | 実用性 → 兵部 | リスク → 刑部
門下省の焦点：必要か欲しいだけか？1ヶ月後もまだ買いたいか？

## 7. 人間関係の判断
人間関係 → 吏部 | 感情 → 工部（体健）| リスク → 刑部 | 社交 → 礼部
門下省の焦点：バイアスをかけて相手を評価していないか？

## 8. レビュー
早朝官に委ねる。日次 / 週次 / 月次 / 四半期 / 年次。

## 9. 健康管理
身体 → 工部 | 時間 → 兵部 | コスト → 戸部 | 規律 → 刑部

## 10. 学習計画
コンテンツ → 礼部 | 実行 → 兵部 | コスト → 戸部 | ネットワーク → 吏部

## 11. 時間管理とエネルギー最適化
エネルギー → 兵部（後勤）| スケジューリング → 兵部（軍令）| ツール → 兵部（装備）| コスト → 戸部 | 習慣 → 刑部 | リズム → 工部
門下省の焦点：本当に時間がないのか、何かを避けているのか？切り捨てようとしているものを実際に気にしている人はいるのか？最後に罪悪感なく休んだのはいつか？

## 12. 家族の重大判断
財務 → 戸部（儲備 + 度支）| 分担 → 吏部（調配）| 関係性 → 吏部（考功）| 実行 → 兵部 | 情報収集 → 兵部 | 法的 → 刑部 | 不測の事態 → 刑部 | 身体/精神 → 工部 | 住居 → 工部
門下省の焦点：誰の声がまだ聞かれていないか？最悪のケースを乗り越えられるか？5年後、やったことを後悔するか、やらなかったことを後悔するか？
