# 场景：大额消费

**路径**：🏃 快车道分析

## 用户消息

```
我想买一台 MacBook Pro M4 Max，顶配大概 60 万日元。现在用的是 3 年前的 M1 MacBook Air，日常开发够用但跑 AI 模型有点慢。主要是看到同事都换了新款，感觉自己也该升级了。
```

## 预期行为

- **Router**：应上报朝廷（金额 > 月收入 10%），建议启用 FINANCE + EXECUTION + GOVERNANCE
- **Planner**：不应启用全 Six Domains（不涉及人际/学习/健康），3 个领域够
- **FINANCE 领域**：应量化 60 万日元占收入/存款的比例，评估财务合理性
- **EXECUTION 领域**：应评估 M1 Air vs M4 Max 的实际性能差距，是否有更高性价比选择
- **GOVERNANCE 领域**：应检查"同事都换了"是否是 FOMO，评估冲动消费风险
- **Reviewer 感性审查**：应质疑"看到同事都换了"这个动机
- **Advisor**：应指出从众心理信号

## 质量检查点

- [ ] Router 没有直接说"买吧"或"别买"，正确上报
- [ ] Planner 没有启用全 Six Domains（启用了 ≤ 4 个领域）
- [ ] FINANCE 领域给出了具体的财务比例
- [ ] EXECUTION 领域分析了实际需求 vs 想要的差距
- [ ] GOVERNANCE 领域或 Reviewer 指出了 FOMO / 从众动机
- [ ] 总评分不应 > 8（存在明显非理性动机）
