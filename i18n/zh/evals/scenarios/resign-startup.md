# 场景：辞职创业

**路径**：⚖️ 完整朝议

## 用户消息

```
我在一家日本公司做软件工程师，年收入 800 万日元，在东京工作了 3 年。最近越来越觉得没意思，想辞职做自己的 SaaS 产品。存款大概有 500 万日元。没有联合创始人，也没想好具体做什么产品，就是觉得应该趁年轻拼一把。
```

## 预期行为

- **Router**：应上报朝廷（涉及职业+财务+不可逆变化），不应直接处理
- **Planner**：应启用全 Six Domains，至少覆盖财务/能力/人脉/执行/风险/生活 6 个维度
- **Reviewer 审规划**：应关注"没想好具体做什么产品"这个关键缺失
- **FINANCE 领域**：500 万日元 ÷ 东京月生活费 ≈ 20-25 个月 runway，但需减去创业启动资金
- **GOVERNANCE 领域**：应检查竞业限制（日本公司常见）、签证影响（如果是工签）
- **EXECUTION 领域**：应指出"没有具体方向"是执行最大障碍
- **INFRA 领域**：应关注创业期间身心健康和生活质量
- **Reviewer 终审**：评分不应全部 ≥ 7（"没想好做什么"是明显短板）
- **Advisor**：应关注"趁年轻拼一把"是否是认知偏差

## 质量检查点

- [ ] Router 没有直接处理，正确上报
- [ ] Planner 启用了 ≥ 5 个领域
- [ ] Reviewer 至少质疑了一个维度
- [ ] FINANCE 领域给出了具体 runway 数字
- [ ] GOVERNANCE 领域提到了竞业限制或签证问题
- [ ] 至少有一个领域评分 < 6
- [ ] 行动项包含"先确定产品方向"
- [ ] Advisor 有实质性进谏（不是正确废话）
- [ ] Auditor 指出了至少一个可改进点
