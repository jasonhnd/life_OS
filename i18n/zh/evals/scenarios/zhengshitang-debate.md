# 场景：政事堂辩论

## 用户消息

```
我拿到一个去新加坡的 offer，年薪翻倍到 1600 万日元等值。但我老婆刚怀孕 4 个月，她父母在东京，我父母在国内。她说她支持我但看得出来不太开心。Offer 要求下个月入职。
```

## 设计意图

让至少两个部门产生方向性矛盾：户部（年薪翻倍 = 强正面）vs 吏部/工部（孕期搬家+远离父母+配偶真实意愿存疑 = 强负面）。这种矛盾应触发政事堂辩论。

## 预期行为

- **丞相**：应上报（多领域+不可逆+影响家人），建议启用全六部或至少 5 部
- **户部**：大概率评分较高（7-9），年薪翻倍是强正面信号
- **吏部**：大概率评分较低（3-6），孕期搬家+远离双方父母+配偶真实意愿存疑
- **工部**：大概率评分较低（3-5），孕期跨国搬家对身心健康风险大
- **刑部**：应指出"她说支持但看得出不开心"是未充分知情同意的信号
- **门下省终审**：应发现评分方向矛盾，建议或触发政事堂
- **政事堂辩论**：
  - 第 1 轮：各方陈述立场
  - 第 2 轮：应出现让步或条件化方案（如"延迟入职 3 个月"、"协商远程过渡期"）
  - 第 3 轮：应收敛为共识点和不可调和分歧
- **中书省整理**：共识与分歧应清晰分开

## 质量检查点

- [ ] 至少 2 个部门评分差距 >= 3 分
- [ ] 门下省终审明确指出了矛盾
- [ ] 政事堂辩论被触发（不是被跳过）
- [ ] 3 轮辩论有递进性：第 2 轮不是重复第 1 轮
- [ ] 第 2 轮或第 3 轮出现了条件化方案（"如果...则..."）
- [ ] 中书省整理中共识和分歧分开列出
- [ ] 最终奏折反映了政事堂辩论结果
- [ ] 行动项包含"与配偶深入对话，确认真实意愿"
