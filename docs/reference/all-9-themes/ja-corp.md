## 文化背景解说

**社長室** 以商务人员的身份说话——简洁、结果导向、使用敬语。**経営企画部** 制定中长期战略。**法務部** 有权审查风险、驳回违反合规的方案——这是本主题的否决节点。**内部監査室** 独立于各部门，直接向取締役会报告。**社外顧問** 不受内部利害关系约束，提供坦率的行为评审。

### 日本职场的直觉

本主题使用的是日本商界每天都在使用的部门名——任何在日本公司工作过的人都能立刻理解这些词：

- **社長室** = 秘书 + 协调核心，日本公司通常是跑腿和总协调的中枢
- **経営企画部** = 每家大企业必有，负责中期经营计划（中計）
- **法務部** = 所有合同、重要决策必经之地
- **経理部**（财务） vs **総務部**（总务 / 行政）是日本企业的标准分工
- **内部監査室** 报告给取締役会而非社長——这是现代日本企业治理的标准做法

### 为什么选这个主题

- 你是日本职场人（サラリーマン / OL）

## 设计约束

- **社長室而非社長** 作为 ROUTER——社長室是日本公司实际接收事务的单位，社長是更高层的象征角色
- **社外顧問作为 ADVISOR** 是日本公司治理的常见安排——独立于内部的利害关系
- **朝礼作为 Start Session 触发词** 是日本公司每日例行的晨会——比"閣議"更贴合企业语境
- **経理部 vs 財務省** 的区别——経理部是企业会计记账口径（更偏记账），而政府主题使用"財務省"更偏宏观财政
- **プロジェクト推進部** 是近年日本企业常见的新型部门名，体现"执行与项目驱动"
