# STRATEGIST 概览 · 人类智慧殿堂

## 一句话定位

STRATEGIST 是你与历史上 93 位最伟大思想家的对话场。它不产出审计报告，不打分，不走"起草-审查-执行"流程。它是纯粹的思考伙伴。

## STRATEGIST vs COUNCIL：到底哪个是哪个？

新用户最容易混淆的两个角色。它们都涉及"多人对话"，但用途完全不同。

| 维度 | STRATEGIST（翰林院 / Advisory Board / 内閣特別顧問） | COUNCIL（政事堂 / Board Meeting / 閣議） |
|------|--------|--------|
| **本质** | 个人思考助手 | 领域冲突调解器 |
| **参与者** | 历史思想家（苏格拉底、老子、马斯克…） | 你系统里的六部（户部、吏部、兵部…） |
| **触发方式** | ROUTER 检测到抽象需求 → 询问你要不要开启 | REVIEWER 检测到领域分数差 ≥3 → 自动触发 |
| **场景** | 价值观困惑、人生方向、身份认同、深度思辨 | "该不该辞职"的决策中户部说不要、兵部说要 |
| **输出** | 对话记录 + 你的顿悟时刻 | 结构化辩论 + 共识/分歧清单 |
| **是否走决策流程** | 否。独立运作 | 是。嵌在"起草-审查-执行"里 |
| **主持人是谁** | 你自己 + STRATEGIST 作为 moderator | REVIEWER |
| **你能否主动召唤** | 能（说"我想找人聊聊"） | 不能。由系统根据分歧自动触发 |
| **结束后归档路径** | `_meta/journal/{date}-strategist-{slug}.md` | 并入 Summary Report |

简单记法：**COUNCIL 解决"做不做"的冲突。STRATEGIST 解决"我是谁"的困惑。**

## 何时该用 STRATEGIST

以下情况是 STRATEGIST 的天然领地：

### 1. 抽象思考，没有具体决策

- "我最近一直在想，自由和稳定到底哪个更重要"
- "我在思考什么叫'好的一生'"
- "创造和重复之间，边界在哪"

这些问题不需要户部算钱、兵部排时间。它们需要**思想深度**。

### 2. 价值观困惑 / 内心矛盾

- "我说我重视家庭，但我总是优先工作——哪个是真的我？"
- "我想自由，但我怕后果——这是懦弱还是理性？"
- "我学了这么多方法论，为什么还是活得很累？"

STRATEGIST 会推荐两类思想家：**与你观点一致的**（让你看到自己思想的谱系）和**对你有益挑战的**（让你看到盲点）。

### 3. 人生方向 / 重大身份转变

- "我该不该辞职去创业"（注意：如果你想要的是财务分析和风险评估 → 走主流程；如果你想问的是"做什么样的人"→ 走 STRATEGIST）
- "40 岁了，我还能不能转行"
- "作为父亲/母亲/儿子/女儿，我的责任边界在哪"

### 4. 想和历史对话，而不是和 AI 对话

有时候你只是想问孔子会怎么看微信朋友圈文化。想和马斯克辩论一场关于风险的看法。想让庄子和尼采在你面前吵起来。

这本身就是价值。STRATEGIST 给你这个空间。

## 何时**不该**用 STRATEGIST

STRATEGIST 不是万能的。以下场景请走主流程：

- **具体决策**："我该不该买这只基金" → 让户部算。
- **执行规划**："这个月怎么安排学 Rust" → 让礼部+兵部出方案。
- **情绪宣泄**："我老板又骂我了" → ROUTER 直接接住情绪，别上升到柏拉图。
- **技术问题**："这段 Python 为什么报错" → ROUTER 直接答。

用 STRATEGIST 处理以上场景，就像用航空母舰运一箱苹果：能做，但浪费。

## 如何触发

三种触发方式：

### 方式 1：ROUTER 主动询问（推荐）

你自然地表达抽象困惑，ROUTER 会检测到并询问：

> 你："我最近一直在想，坚持和放手的边界在哪里"
>
> ROUTER："要不要召开翰林院，和几位历史思想家聊聊？" / "Would you like to activate the STRATEGIST and dialogue with some historical thinkers?"

你说"要"/"yes"/"好"，STRATEGIST 就会被召唤。

### 方式 2：你直接点名

> 你："召开翰林院" / "启动 STRATEGIST" / "我想和苏格拉底聊聊"

ROUTER 会直接交给 STRATEGIST。

### 方式 3：在主流程中途叫停

主流程进行中你突然觉得"等等，这不是决策问题，是价值观问题"：

> 你："停一下，我想先和几个思想家聊聊这个"

ROUTER 暂停主流程，启动 STRATEGIST。聊完后你可以选择继续主流程或直接结束。

## 会话流程（5 步）

STRATEGIST 上线后会走固定的 5 步：

1. **理解目的** — "你今天想探讨什么问题？"（不急着列清单）
2. **展示索引 + 推荐** — 基于你说的话，推荐 2-3 位思想家 + 对话模式（一对一 / 圆桌 / 辩论）
3. **启动思想家** — 每位作为独立 subagent，深度扮演（不是"关于苏格拉底的知识"，是"苏格拉底本人在和你说话"）
4. **主持对话** — STRATEGIST 作为主持人协调，你可以随时插话、切换模式、换人
5. **结束仪式** — 每位思想家临别赠言 → STRATEGIST 总结旅程 → 归档到 `_meta/journal/`

具体模式怎么工作、怎么切换，见 [三种对话模式](three-dialogue-modes.md)。

## 重要原则

### 1. 独立 context，绝不串台

这是 HARD RULE。每位被召唤的思想家是独立 subagent，只看到话题和自己的角色，看不到其他思想家的完整输出（只能看到对方发言的摘要）。这保证每位的"独立思考"是真实的。

### 2. 深度扮演，不出戏

苏格拉底不会说"作为 AI，我认为…"。他会说"🔎 你刚才说'我没得选'，这个词让我很感兴趣——我们来拆拆看"。

如果某位思想家开始"破功"，说明 subagent 启动有问题，可以让 STRATEGIST 重启。

### 3. 不给直接答案，引导你自己找答案

苏格拉底不会告诉你"该辞职"。他会问到让你自己说出"其实我早就想清楚了"。马斯克不会给你一个商业计划，他会挑战你"这件事不做的代价是什么"。

**结论必须是你自己的。**

### 4. 可以点名列表外的人

93 位是推荐，不是限制。你可以召唤：
- 张三丰
- 鲁迅（已在列表）+ 胡适（列表外）
- 你父亲 / 你奶奶（是的，你认识的真实人物也行）
- 某本书里的某个角色

STRATEGIST 会用同等深度扮演。前提是这个人物有足够的真实素材可供扮演（史料、著作、言行记录）。虚构角色若有足够设定也可以。

## SOUL.md 的影响

如果你的 second-brain 里有 SOUL.md（你的个性档案），STRATEGIST 会读它。比如：
- SOUL 记录"自由 vs 稳定"的张力 → 会推荐塞涅卡（约束中的自由）
- SOUL 记录你的英雄是乔布斯 → 会考虑推荐乔布斯直接对话
- SOUL 记录你有"过度理性"倾向 → 可能推荐一个情感派思想家挑战你

详见 [SOUL 匹配逻辑](soul-matching.md)。

## 一个完整例子

```
你：我最近一直在想，我做得这么多，到底图什么

ROUTER：听起来不是一个具体决策，是个更深的问题。要不要召开翰林院？

你：好

STRATEGIST（翰林院）：你想探讨什么？是"图什么"这个词背后的焦虑，
    还是想弄清楚你真正在意什么？

你：后者吧。我觉得我好像在追别人定义的成功

STRATEGIST：基于你的话，我推荐三位：
    1. 庄子——会问你"鱼之乐"，让你看清"别人的标准"这件事本身荒诞
    2. 马可·奥勒留——斯多噶派，会帮你区分"你能控制的"和"不能控制的"
    3. 稻盛和夫——会从经营者视角谈"为何而活"
    
    模式推荐：先和庄子一对一（20 分钟），然后圆桌加入另外两位。
    
    同意吗？

你：同意

→ STRATEGIST 启动庄子 subagent
→ 庄子："你刚才说'别人定义的成功'，先不急着定义'别人'是谁——
       我先问你：你能分清你说这话时，心里有没有一个很具体的人？"
→ …

（对话持续）

你：算了，我想清楚了

STRATEGIST：好。让三位说句临别话——

庄子：鱼之乐，不在水之深，在是否是鱼。
马可·奥勒留：做你今天就能做的那部分。
稻盛和夫：利他，便不会空虚。

我的总结：
📝 旅程：从"图什么"的焦虑 → 识破"别人定义的成功"的幻象 → 
       到达"今天能做的那部分"的落点
💡 关键顿悟：你说过一句"其实我一直知道我要什么，只是不敢" — 
       这是今天最重要的一句话
❓ 悬而未决：那个"不敢"，下次我们可以专门探讨

归档到 _meta/journal/2026-04-20-strategist-meaning-of-doing.md
```

## 常见疑问

**Q：一次最多召几位？**
A：圆桌最多 4 位，辩论只能 2 位。一对一就是 1 位。超过会导致每个人发言深度不够，失去意义。

**Q：一场对话一般多久？**
A：一对一 15-30 分钟实际对话；圆桌 30-60 分钟；辩论 20-40 分钟（3 轮）。但 STRATEGIST 不会催你，你说"够了"就结束。

**Q：结束后会自动走主流程的 Summary Report 吗？**
A：不会。STRATEGIST 是独立路径，不走起草-审查-执行，也不触发 AUDITOR / ADVISOR。只归档到 journal。

**Q：和主流程的关系？**
A：互补。有些决定，你需要先想明白"我是谁"（STRATEGIST），再算"怎么做"（主流程）。两条路可以前后接力。
