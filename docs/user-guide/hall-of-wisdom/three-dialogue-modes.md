# 三种对话模式 · 一对一 / 圆桌 / 辩论

STRATEGIST 支持三种对话模式。选对模式比选对思想家更重要——模式决定了思考的形状。

## 快速对照

| 模式 | 人数 | 本质 | 适合场景 | 典型时长 |
|------|------|------|----------|----------|
| 🎭 一对一 | 1 位 | 深度挖掘 | 困惑具体、需要反复追问 | 15-30 分钟 |
| 🪑 圆桌 | 2-4 位 | 多视角对照 | 困惑宽泛、想看不同角度 | 30-60 分钟 |
| ⚔️ 辩论 | 2 位（对立面） | 强制取舍 | 选择题、二选一、想看清对立理据 | 20-40 分钟 |

## 🎭 一对一（One-on-One）

### 如何工作

STRATEGIST 启动选定的思想家作为独立 subagent，然后**退到幕后**。你和思想家直接对话，不经过第三者。STRATEGIST 只在你对"他"说话时才出现（比如"我想换一个人聊"）。

对话是来回的，像真的聊天：
- 思想家问你一个问题
- 你回答
- 思想家基于你的回答再追问 / 反驳 / 引导
- 你继续
- …

### 何时选这个

**你的困惑很具体，只是需要被挖深。**

例子：
- "我知道我在躲避什么事，但我不知道是什么" → 和苏格拉底一对一（他就是干这个的）
- "我对财富的焦虑从哪来" → 和弗洛伊德或荣格一对一
- "我想学某种修行" → 和铃木大拙或慧能一对一

### 如何触发

```
STRATEGIST：我推荐你和 X 一对一聊

你：好
```

或你直接指定：
```
你：我想和老子一对一聊
```

### 什么时候该退出

一对一聊着聊着，你可能会发现：
- "这个问题其实有两面，我希望有人反驳" → 切换到辩论
- "我想再听听其他角度" → 切换到圆桌

切换方式见下文"中途切换"章节。

## 🪑 圆桌（Roundtable）

### 如何工作

STRATEGIST 把 2-4 位思想家召到同一个"圆桌"。每位独立 subagent，互相看不到彼此的完整思考，只能看到发言**摘要**。

流程：
1. STRATEGIST 介绍议题，设定发言顺序
2. 第 1 位发言（回应议题和你之前的话）
3. STRATEGIST 把这位的发言核心摘要传给第 2 位
4. 第 2 位发言（可能呼应、补充、或反对前者）
5. 每轮结束后，STRATEGIST 做小结："这里他们一致…这里他们分歧…"
6. 你随时可以插话、指定某位深入、或换话题

### 何时选这个

**你的困惑是开放式的，需要多角度光照。**

例子：
- "好的一生是什么样" → 圆桌：孔子 + 亚里士多德 + 尼采 + 稻盛和夫
- "创造 vs 稳定，哪个更重要" → 圆桌：达芬奇（创造） + 本杰明·富兰克林（纪律） + 庄子（超脱）
- "我该怎么思考衰老和死亡" → 圆桌：塞涅卡 + 释迦牟尼 + 海伦·凯勒

### 推荐搭配逻辑

圆桌的**张力来自视角差异**。好的组合有：
- 东方 + 西方（老子 + 苏格拉底）
- 理性 + 情感（康德 + 陀思妥耶夫斯基）
- 行动派 + 反思派（马斯克 + 庄子）
- 入世 + 出世（孔子 + 释迦牟尼）

避免把观点高度重合的凑一桌（比如康德 + 黑格尔 + 维特根斯坦，德语圈理性主义三连——你听到的会像一个人说三遍）。

### 如何触发

```
你：我想听几个角度

STRATEGIST：我推荐这四位组圆桌 …（列出 + 解释）
```

或：
```
你：圆桌一下，老子、尼采、马斯克
```

## ⚔️ 辩论（Debate）

### 如何工作

两位思想家，**对立立场**，就一个命题正面对决。3 轮：

1. **第 1 轮 · 陈述**：A 方陈述立场（为什么支持/反对）→ STRATEGIST 把 A 方核心论证传给 B 方 → B 方回应
2. **第 2 轮 · 反驳**：A 方反驳 B 方的论点 → B 方反驳回去
3. **第 3 轮 · 终陈**：双方最后陈述

结束后：
- STRATEGIST 总结：双方核心论证、根本分歧点在哪、你可以从各方各取什么
- **你做最终判决**——不是 STRATEGIST，不是思想家。你。

### 何时选这个

**你面临一个选择题，且你觉得自己被某一方吸引，想看清对立方的最强论证。**

例子：
- "要不要辞职创业" → 辩论：马斯克（去做） vs 巴菲特（耐心等完美机会）
- "要不要把这段关系结束" → 辩论：斯多噶派的马可·奥勒留（聚焦你能控制的） vs 孔子（重承诺、重关系的修复）
- "躺平还是卷" → 辩论：老子（无为） vs 富兰克林（纪律即自由）

### 设定命题的艺术

辩论命题要**可辩**——即要能听出两边都有道理。
- ✅ "30 岁了，还在打工是浪费时间"（可辩：马斯克支持 / 稻盛和夫反对）
- ❌ "我是一个好人"（不可辩，太主观）
- ❌ "地球是圆的"（不可辩，单一答案）

### 如何触发

```
你：我想让 X 和 Y 辩一场

STRATEGIST：好。命题是什么？
```

或 STRATEGIST 检测到你有明显选择题倾向时会主动提议：
```
STRATEGIST：你这个问题其实是两难抉择。要不要让 X 和 Y 辩一场？
```

## 中途切换

你**不需要**一开始就选对模式。STRATEGIST 支持随时切换：

### 切换指令（自然语言即可）

| 你说 | STRATEGIST 动作 |
|------|-----------------|
| "加一个 XX" | 启动 XX subagent，切换到圆桌 |
| "让 XX 和 YY 辩一下" | 切换到辩论模式 |
| "换成 ZZ" | 结束当前 subagent，启动 ZZ |
| "我想只和 XX 聊" | 退回一对一 |
| "先停一下，我想想" | 所有思想家暂停，STRATEGIST 待命 |
| "让 XX 先别说了" | 那位暂时沉默，其他继续 |

### 切换的信号

什么时候该切？听你自己的感觉：

- 一对一时，你觉得"我想听反面" → 加人 / 变辩论
- 圆桌时，某位特别打动你 → 退到和他一对一
- 圆桌太散，听不到重点 → 结束圆桌，选一位深挖
- 辩论时，感觉双方都在说废话 → 换人 / 加个"调停者"
- 情绪上头 → 暂停，让 STRATEGIST 帮你喘口气

## 结束仪式

三种模式，结束仪式**相同**——这是 HARD RULE。

### 触发结束

你说任意一句：
- "我想清楚了" / "够了" / "谢谢"
- "可以结束了" / "先这样吧"
- "I've figured it out" / "done" / "thanks"

### 3 个步骤

**1. 临别赠言（Parting Words）**

每位在场的思想家各说**一句**最后的话。**必须在角色里**——不是总结陈词，是"这个人此刻对你最想说的一句话"。

例子：
- 庄子：鱼之乐，不在水之深，在是否是鱼。
- 马斯克：Make sure the next 6 months matter.
- 孔子：己所不欲，勿施于人——包括你自己。

**2. STRATEGIST 总结（以主持人身份，不是任何角色）**

三段结构：
- 📝 **旅程**：思考从哪里开始 → 关键转折点 → 到达哪里
- 💡 **关键顿悟**：**你自己**说过的最有分量的话（不是思想家说的）
- ❓ **悬而未决**：还没回答的问题（如果有）

**3. 归档**

写入 `_meta/journal/{date}-strategist-{slug}.md`：
- 前置元数据：`type: journal`, `journal_type: strategist`
- 内容：参与思想家、议题、旅程、顿悟、临别赠言

这个档案会在下次 Start Session 的时候被 RETROSPECTIVE 读到，作为你的思考历史一部分。

## 常见错误 / 反模式

### ❌ 一次叫 5 个人开圆桌
深度会稀释。每个人发言 2 分钟就过去了，没人说深。最多 4 位。

### ❌ 让 STRATEGIST 也参与对话
STRATEGIST 是主持人，不是思想家。让它"代表某派观点"就串台了。想要另一个视角，直接叫一位思想家。

### ❌ 跳过结束仪式直接切会话
会话终止 ≠ 思考终止。临别赠言和 STRATEGIST 总结是让你**真的**带走东西的那一步。不要省。

### ❌ 辩论命题模糊
"我要不要开心一点" 这种不能辩。辩论命题应该**可证伪**、**有对立立场**。

### ❌ 把辩论当成"谁赢了"
辩论不是比赛。重点是**看清两边的最强论证**，然后**你**做判断。如果你想让 STRATEGIST 告诉你谁赢了，你就错失了辩论的价值。

## 一句话记忆

- **一对一**：挖一口井
- **圆桌**：开一扇窗，看全景
- **辩论**：把两边都摆上台，强迫你选
