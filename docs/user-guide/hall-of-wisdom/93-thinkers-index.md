# 93 位思想家索引 · 18 个领域

人类智慧殿堂收录 93 位历史上最具影响力的思想家，按 18 个领域分类。本文是完整索引，每位简注其领域和方法。

> 这个名单是**推荐**，不是**限制**。你可以召唤名单之外任何真实或有足够设定的人物，STRATEGIST 会用同等深度扮演。见底部"如何请求列表外的人"。

## 🔬 1. 科学与认知（Science & Cognition）· 6 位

| 思想家 | 领域 | 核心方法 |
|--------|------|----------|
| **牛顿**（Newton） | 经典物理、光学、微积分 | 用数学精确描述自然，"如果我看得更远，是站在巨人肩上" |
| **爱因斯坦**（Einstein） | 相对论、量子物理 | 思想实验（如果我追光会怎样？），打破既定框架 |
| **费曼**（Feynman） | 量子电动力学 | "如果你不能用简单语言解释，就是你没真懂"——还原根本 |
| **达尔文**（Darwin） | 进化论、生物学 | 长期观察 + 大胆假设，看到物种间的连续性 |
| **图灵**（Turing） | 计算理论、密码学 | 抽象机器模型、"能否计算"的数学定义 |
| **居里夫人**（Marie Curie） | 放射化学 | 用尽一切证伪假设，极致的实验严谨 |

## 🏛️ 2. 哲学与推理（Philosophy & Reasoning）· 8 位

| 思想家 | 领域 | 核心方法 |
|--------|------|----------|
| **苏格拉底**（Socrates） | 古希腊哲学 | 助产术——通过追问让你自己发现矛盾 |
| **柏拉图**（Plato） | 理念论、政治哲学 | 对话录、比喻（洞穴、战车）、追问理想形态 |
| **亚里士多德**（Aristotle） | 逻辑、伦理、政治 | 分类、中庸、四因说、实践智慧（Phronesis） |
| **康德**（Kant） | 批判哲学 | "普遍化准则"——你希望所有人都这样做吗？ |
| **黑格尔**（Hegel） | 辩证法、历史哲学 | 正-反-合，矛盾推动进步 |
| **尼采**（Nietzsche） | 存在主义先声 | 挑衅、重估一切价值、"杀死内心的骆驼" |
| **维特根斯坦**（Wittgenstein） | 语言哲学 | "能说的都能说清，不能说的请保持沉默" |
| **萨特**（Sartre） | 存在主义 | "存在先于本质"——你不是什么，是你选择成为什么 |

## 如何请求列表外的人

**这 93 位是起点，不是终点。** 你可以在任何时候要求召唤：

### 1. 其他历史人物

- "我想和胡适聊"（民国思想家，列表没有）
- "召来王阳明和曾国藩辩论"（王阳明在列表，曾国藩在列表，完全可以）
- "启动 Linus Torvalds"（近现代程序员、开源先驱）
- "Vimalakirti（维摩诘）"（佛教经典人物）

前提：此人有足够的**真实素材**——著作、言行、书信、传记。这样 STRATEGIST 才能真实扮演，而不是编。

### 2. 你认识的真实人物

- "我想和我去世的祖父聊一次"
- "启动我大学时的李老师——我会描述他"

STRATEGIST 会让你先描述这个人（关键言行、性格、他会怎么说话），然后尽力扮演。效果取决于你能描述多准。

### 3. 文学/历史角色（有足够设定的）

- 诸葛亮的《出师表》版 vs 历史诸葛亮 vs 三国演义诸葛亮——可以，指定版本
- 《红楼梦》的贾宝玉
- 《基督山伯爵》的爱德蒙·唐泰斯

### 4. 抽象角色（慎用）

- "未来的你"——可以，但本质上是你自己想象
- "作为某个职业的代言人"——可以，但不如直接叫这个职业里的真人

### 如何指定

最简单：直接说名字。
```
你：我想和 XX 聊
```

或加说明：
```
你：启动 XX，我想和他讨论 YY
```

STRATEGIST 会快速确认：
- 这个人物的主要时期/版本（如果有多个）
- 对话的模式（一对一 / 加入现有圆桌 / 替换某位）

## 如何挑人（给不知道选谁的你）

### 规则 1：先定问题的**性质**

| 问题是什么 | 去哪些领域挑 |
|----------|-----------|
| 我想看透人性 | 🎭 人性与洞察、🌏 东方思想、🏛️ 哲学 |
| 我在做决定 | 💰 商业、⚔️ 战略 |
| 我被情绪困住 | 🧘 心智与修行、🎭 人性与洞察 |
| 我在学东西 | 🔬 科学、📐 系统与纪律、🔢 数学与逻辑 |
| 我面临逆境 | 🔥 逆境与反脆弱、🧘 心智与修行 |
| 我思考"活着的意义" | 🏛️ 哲学、🌏 东方、🔥 逆境 |

### 规则 2：同领域不要挑太近

挑苏格拉底 + 柏拉图 + 亚里士多德——一个师承链条，观点太像。
挑苏格拉底 + 老子 + 马斯克——古希腊追问 + 东方无为 + 第一性原理，有张力。

### 规则 3：跟着 STRATEGIST 的推荐

如果你真的不确定，告诉 STRATEGIST 你的问题，让它推荐。它会基于：
- 你说的话的关键词和情绪
- 你的 SOUL.md（如果存在）
- 问题的性质（值观 / 决策 / 修行 / 学习）

然后给你 2-3 位候选，你点头就开始。
