## 何时选哪个

### 🏛️ 明治政府 — 选这个如果...

- 你喜欢**明治の気概**——日本近代化启动时那种"非做不可"的志气
- 你欣赏明治时期制度设计的精妙：元老制度、枢密院、大蔵省 … 很多**原创设计**
- 你做**人生级重大决策**，希望有"歴史の重み"
- 你懂日语历史，看到"大蔵省"会感到亲切（今天已叫"財務省"，但"大蔵"这个词仍在日语日常使用中存活）

**特别之处**：明治的"元老"制度是 ADVISOR 的完美对应——伊藤博文、山縣有朋、西園寺公望这些元老没有正式职务，但可以对首相直接说"最近の方向は間違っている"。这种**不受约束的直言**正是 ADVISOR 的本质。

### 🏛️ 霞が関 — 选这个如果...

- 你熟悉**現代日本中央省庁**（看新闻、工作相关）
- 你看到"財務省"立刻知道管钱的，看到"法務省"立刻知道管合规
- 你做**政策类 / 合规类 / 大方向**决策，政府外壳最契合
- **最无学习成本**：日本の省庁は 6 つの生活領域に自然にマッピングされる

### 🏢 企業 — 选这个如果...

- 你在**日本企业**工作，社長室 / 経営企画部 / 法務部 是你每天的词
- 你做**商业 / 職業 / 人事**决策，企业外壳最契合
- 你想要**敬語のビジネス調**——专业、结果重视、有分寸
- **日本の会社員にとって最も自然なフレームワーク**

## 文化背景详解

### 🏛️ 明治政府

**歴史根**：明治維新（1868-1912）によって誕生した近代日本の統治機構。重要点：

- **内閣総理大臣** 明治政府の首長
- **参議** 国家の方向性を策定する高位役職（PLANNER）
- **枢密院** 天皇の最高諮問機関として、憲法や重大政策を審査する**実質的な拒否権**を持つ（REVIEWER）
- **大蔵省** 明治財政の代名詞——今日なお日本人が財務を語る際の古典的な呼称
- **工部省** 鉄道、電信、鉱山を統括し、明治の近代化そのものを象徴
- **会計検査院** 1880 年に設立、内閣から独立して天皇に直接報告——日本の独立監査制度の原点（AUDITOR）
- **元老** 伊藤博文、山縣有朋、西園寺公望ら——正式な役職を持たないが、首相に「最近の方向は間違っている」と直言できる（ADVISOR）

**Life OS での意義**：元老の制度は ADVISOR と完全に一致する。正式な権限なしに、影響力だけで「違うぞ」と言う存在。この制度設計は明治日本の独創だった。

### 🏛️ 霞が関

**体制根**：現代日本の中央省庁システム。重要点：

- **内閣官房長官** 政府のスポークスパーソン兼コーディネーター——案件を適切な省庁に割り振る中央人物（ROUTER に最適）
- **内閣法制局** 政策案を法的観点で審査し、拒否する権限を持つ（REVIEWER）
- **会計検査院** 憲法上独立し、すべての政府活動を監査する（AUDITOR）
- **内閣参与** 総理への特別助言者、部門利害から自由（ADVISOR）

**最大の強み**：日本の省庁構造（財務省、法務省、文部科学省など）は 6 つの生活領域に**ほぼ完璧にマッピング**される。日本人ユーザーが「財務省」を見た瞬間、「これはお金の話だ」と理解する——説明不要。

### 🏢 企業

**組織根**：日本企業の一般的な部門構造。重要点：

- **社長室** ビジネスパーソンとして語る——簡潔、結果志向、敬語を使う
- **経営企画部** 中長期戦略を策定（PLANNER）
- **法務部** リスクを審査し、コンプライアンス違反があれば差し戻す権限（REVIEWER）
- **内部監査室** 各部門から独立し、取締役会に直接報告（AUDITOR）
- **社外顧問** 社内の利害関係に縛られず、率直な行動レビューを提供（ADVISOR）

**最も自然な日常語**：日本のビジネスパーソンが毎日使う部門名を採用。仕事の意思決定を考える際に最もスムーズ。

## 実戦例

### 例 1：明治政府で重大決定

```
あなた：会社を辞めて起業しようと思っている。

内閣総理大臣：🏛️ 国家の大事に相当する決定です。参議に策定、
             枢密院に審査、各省に評価を命じます。

参議：📜 参議策定書
    議題：退職と起業
    背景：...
    案 A：即時退職、フルタイム起業
    案 B：半年間の副業と貯蓄
    ...

枢密院：🔍 審査する。
    案 A の財政前提が楽観的すぎる——差し戻す。
    案 B に条件付き承認：12 か月分の運転資金を要件とする。

（各省逐次報告）

内閣総理大臣：📋 閣議決定：退職と起業
     総合評価 7/10 —— 実現可能、ただし財政準備がより厳密に必要
     
     必須対応：
     - 大蔵省：運転資金 12 か月未満では始動しない
     - 枢密院：書面による撤退条件を要求
     
     御前会議（COUNCIL）が召集された：
     大蔵省と陸軍省の意見が分かれた。結論：初期投資を半減。
     
     元老進言：「慌てるな。明治の元勲も急がず遠くを見た」
```

```
あなた：50 万円を ETF に投資したい。

内閣官房長官：🏛️ 財務分析のみですか、それとも完全な意思決定ですか？

あなた：完全な意思決定で。

[完全フロー実行]

内閣官房長官：📋 閣議決定書：50 万円 ETF 投資
     総合評価 4/10 —— 現配分比率では推奨しない
     
     必須対応：
     - 財務省：ポートフォリオ全体のリスク予算の 5% を超過
     - 法務省：書面による損切り基準なし
     
     閣議（COUNCIL）結論：
     財務省と内閣参与の意見不一致。決定：15 万円以下に減額。
```

## 三者間の切り替え

```
あなた：テーマ切り替え

システム：🎨 現在：霞が関。新しいテーマをお選びください：[g/h/i]

あなた：g

システム（新テーマの語調で）：
    🏛️ 「明治政府」に切り替えました。内閣総理大臣、就任しました。
```

**三つの日本語テーマ間で切り替え**：言語は変わらず（日本語のまま）、表示名だけが変わる。中国語・英語テーマに切り替えれば言語も変わる。

## 反模式 · やってはいけないこと

### ❌ 明治政府で古文を使う
「臣、拝して申し上げ奉る…」——違う。テーマは「明治の気概。簡潔かつ毅然。現代語で書く」と明記されている。志は高く、言葉は現代で。

### ❌ 霞が関でカジュアル語
「ってかさぁ…」——違う。霞が関は「ビジネス丁寧語、です・ます調」。プロフェッショナルに。

### ❌ 企業で上から目線
「お前は…」「これダメだろ」——違う。ビジネス敬語、結果重視、でも相手への敬意は保つ。

### ❌ 日英混在
日本語テーマを選んだら "CFO" / "roadmap" / "KPI" は使わない。"経理部" / "ロードマップ" / "重要指標" を使う。

## ADVISOR は「元老 / 内閣参与 / 社外顧問」

日本語テーマにおける ADVISOR は、それぞれ異なる「直言者」の形：

- **元老（明治政府）**：正式な権限なしに、総理に「違う」と言える歴史的存在。伊藤博文のイメージ。
- **内閣参与（霞が関）**：総理への特別助言者。部門利害から自由。
- **社外顧問（企業）**：社内の政治から独立した立場。率直で構わない。

共通点：**利害関係がないから本音を言える**。これが ADVISOR の本質。
